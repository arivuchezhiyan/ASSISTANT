# Work Packs Spec

## Packs
1. Coding Pack: repo analysis, refactor, test, validate.
2. Unreal Pack: detect project, scaffold actor/module, build-fix loop.
3. Ops Pack: system commands with policy guardrails.
4. Security Pack: permissions, allowlists, audit traces.
5. Memory Pack: ingestion, retrieval, retention, validation.
6. Reliability Pack: retries, cache, fallback, recovery gates.

## Shared Contract
- Input schema
- Required tools
- Success criteria
- Rollback and idempotency hints

## Acceptance
- Each pack runnable as a standalone workflow.
- Each pack emits machine-readable result envelope.

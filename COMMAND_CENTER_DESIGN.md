# Command Center Design

## Purpose
Single operational dashboard for health, costs, reliability, and readiness gates.

## Panels
- Runtime health (CPU/RAM/governor mode)
- Session throughput and latency
- Tool reliability and recovery outcomes
- Memory/RAG quality indicators
- Safety and permission events

## KPIs
- Turn success rate
- Tool failure recovery rate
- Mean/95p latency
- Cost per session
- Gate pass/fail history

## Acceptance
- Real-time updates without blocking query path.
- Exportable reports for readiness and go-live documents.
